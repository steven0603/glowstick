# GlowStick X 證券投資信託基金

空間叉叉投資信託(股)公司｜競賽期間：2026/3/22 ～ 2026/5/29

低軌衛星（LEO）及航太通訊供應鏈主題基金，14 檔台股 + 現金，初始資金 $100M USD。

---

## 環境需求

- Python 3.10 以上
- 網路連線（Yahoo Finance 抓取即時資料）

---

## 安裝

```bash
git clone <repo-url>
cd glowstick
pip install -r requirements.txt
```

---

## 啟動

```bash
python3 main.py
```

**首次啟動**會自動偵測是否已初始化。若資料庫為空，請選擇 `0` 進行初始化。

**非首次啟動**會自動補齊上次到今日所有缺漏資料，無需手動更新。

---

## 功能選單

| 選項 | 功能 |
|------|------|
| 1 | 更新今日價格 & 計算淨值 |
| 2 | 查看投資組合（持股表格）|
| 3 | 查看 TAIEX 報酬表 |
| 4 | 生成每日 Markdown 紀錄 |
| 5 | 繪製報酬率折線圖 |
| 6 | 計算 Alpha（完整過程）|
| 7 | 下委託單（買/賣）|
| 8 | 查看待執行委託 |
| 9 | 查看交易紀錄 |
| 0 | 重新初始化系統（重算歷史）|

---

## 多人協作說明

資料庫 `portfolio.db` 存於 repo 中，每次有以下操作後請 commit 並 push：

- 新增交易（選項 7）
- 完成每日更新後

其他組員執行 `git pull` 即可同步最新資料。

```bash
# 有人更新資料後
git add portfolio.db
git commit -m "update: YYYY-MM-DD 資料更新"
git push

# 其他組員
git pull
python3 main.py
```
